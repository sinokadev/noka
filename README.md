# noka

리눅스용 게이밍 유틸리티

# 기능

 - replay

# 빌드방법

`build.sh`를 실행한다.


# 사용방법

주의: ffmpeg이 필요합니다. (ffmpeg이라는 명령어가 존재해야합니다. 프로그램은 내부적으로 쉘 명령어를 실행합니다.)

`noka replay enable`: `~/noka/replay`가 자동으로 생성되며, 모든 모니터를 녹화합니다.(소리 포함) 또한 실행시 한번 옛날 리플레이(아래 참고)가 삭제됩니다. 
`noka replay ps`: 리플레이가 실행중인지 확인합니다. 또한 실행중이라면 pid를 표시합니다.
`noka replay disable`: 실행중인 리플레이가 종료됩니다. 녹화중인 리플레이는 자동으로 저장됩니다.  
`noka replay save -c (N)`: 최근 N개의 리플레이를 저장합니다(저장한 리플레이는 ~~_SAVED.mp3등과 같이 생김.). 저장은 몇초정도 걸릴수 있으며, 그 동안은 녹화되지 않습니다.  
`noka replay delete`: 옛날 리플레이(아래 참고)를 삭제합니다.

기본적으로 60일 전의 리플레이는 삭제되며(저장된 리플레이 제외), 3분 단위(혹은 그보다 더 짧은)로 영상이 잘려 보관됩니다. 사용자에 따라 적절히 코드를 수정하여 이 값을 변경할수 있습니다.

# 주의

제작자가 C언어를 잘하지 않습니다. 따라서 사용중 버그나 리플레이 유실 등 문제가 발생할수 있으며, 이슈로 제보해주신다면 수정하겠지만 저에게 법적 책임은 없습니다.
